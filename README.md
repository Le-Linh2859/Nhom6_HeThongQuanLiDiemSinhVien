🎓 HỆ THỐNG QUẢN LÝ ĐIỂM SINH VIÊN (.NET)

📌 Giới thiệu

Hệ thống Quản lý điểm sinh viên được xây dựng nhằm hỗ trợ nhà trường, giảng viên và sinh viên trong việc quản lý, theo dõi và đánh giá kết quả học tập một cách hiệu quả, minh bạch và chính xác.

Dự án được phát triển trong khuôn khổ học phần Lập trình .NET, hướng tới việc áp dụng các công nghệ hiện đại trong phát triển phần mềm quản lý

⚙️ Công nghệ sử dụng
Ngôn ngữ: C# (.NET)
Giao diện: WinForms + Guna.UI2 / Krypton Toolkit
Biểu đồ: Live Charts
Cơ sở dữ liệu: SQL Server / MySQL
ORM: Entity Framework / Dapper
Bảo mật: Mã hóa mật khẩu (Hashing)

🏗️ Kiến trúc hệ thống

Hệ thống được thiết kế theo mô hình:

3-tier architecture
Presentation Layer (UI)
Business Logic
Data Access Layer (DAL)
